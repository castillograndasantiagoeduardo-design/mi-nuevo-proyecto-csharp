Presentacion By: Santiago Castillo

Inducción - Primer contacto con C#

1. ¿Qué es C#? Lenguaje multiparadigma desarrollado por Microsoft para .NET, seguro y orientado a objetos.

2. ¿Qué es una aplicación de escritorio? Software local que interactúa directamente con los recursos del Sistema Operativo.

3. ¿Qué es Visual Studio? El IDE (Entorno de Desarrollo Integrado) profesional para construir soluciones .NET.

4. Crear un proyecto: Selección de plantilla "Windows Forms App" para iniciar la interfaz gráfica.

5. Entorno de Visual Studio: Explorador de soluciones, Cuadro de herramientas y Ventana de propiedades.

6. El formulario (Form): Clase base que representa la ventana física de la aplicación.

7. Controles y 8. Propiedades

  • Label: Texto descriptivo. (Propiedad: Text)
  • TextBox: Entrada de usuario. (Propiedad: Name)
  • Button: Activador de acciones. (Propiedad: Size)

9. Eventos y 10. MessageBox

Programación dirigida por eventos: el código se ejecuta cuando el usuario hace clic. MessageBox es la herramienta principal para notificar estados.

MessageBox.Show("¡Primera aplicación funcional!");

Lógica y Formularios Funcionales

1. Repaso: Consolidación de controles y diseño de Forms.

2. Variables y Tipos: int, double, string, bool.

3. Conversión de Datos: Métodos Parse y Convert.

4. Condicionales: Control de flujo con IF/ELSE.

5. Ciclos básicos: Estructuras For y While.

6. Validación: Evitar errores de tipo de dato o campos vacíos.

10. Organización del código: Uso de regiones, comentarios técnicos y nombres descriptivos para mantener el código legible.

7. Controles Extra

ComboBox, RadioButton, CheckBox.

8. Manejo de Errores

Bloques Try / Catch para prevenir excepciones.

9. Navegación

Instanciación y uso de múltiples formularios (.Show()).
